Oracle Certified Professional

Posted on 2019-08-21 | In java

终于拿到证书了，非常开心，虽然分数不好但是朋友说过了就是过了，做人要嚣张点，想想确实有道理，所以这次晒个大图嚣张一下。

之前考完1Z0-808之后就开始读Thinking in Java备考809，这书看的我有点怀疑人生，换成Head first Java之后看的就非常舒爽，然后自信满满的把考试订了，弄了套模拟题，挂的有点惨，毕竟讲的东西有点浅。
最终我选的方案是亚马逊弄了本Jeanne Boyarsky的OCP学习指南，OCP: Oracle Certified Professional Java SE 8 Programmer II Study Guide。这书也是相当不错的，厚度适宜，文笔幽默，通俗易懂，最大的问题是书里有很多小错误，得去官网对着勘误表自己校对，这书这么多年来放着这么多错不纠正还只有第一版，确实是有点厉害的。
看完书之后就可以做做模拟题感受一下了，推荐度比较高的是enthuware的，之前准备808的时候在他们那买的模拟题，便宜大碗，两次考试考下来的感觉是一致的：这模拟题难度和实际考试的差不多，覆盖的范围也差不多，但具体到考点上就没多大关联了，背题对实际考试没有任何意义。
结果来说模拟题和实际考试的分数都很微妙，能一次过是真的略开心。然而菜鸡还是菜鸡，通过这次考试愈发认识到自己多菜了。

Oracle Certified Associate
