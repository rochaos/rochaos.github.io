Tips

  • main() method is entry point of all Java applications, whenever a Java application is started, the main() method is the first method to be called;
  • all statemtns in Java must end with a semi-colon;

Variables and Operators

8 Basic data types (predefined) / Primitive data types

  • For storing integers
    • byte (-128 to 127), use 1 byte (2^8) of storage space.
    • short (-32768 to 32767), use 2 byte (2^16) of storage space.
    • int (-2^31 to 2^31-1), use 4 byte (2^32) of storage space, most commonly used data type
    • long (-2^63 to 2^63-1), use 8 byte (2^64) of storage space, need to add suffix ‘L’ to end of the number, rarely used.
  • For storing floating point number
    • float (-3.40282347E38 to 3.40282347E38), use 4 byte of storage space, as precision of about 7 digits (1.2345678), need to add suffix ‘F’ to the end of number.
    • double (-1.79769……E308 to 1.79769….E308), use 8 byte of storage space, with precision of about 15 digits, by default Jave treat floating point number as double.

  • Other primitive data type

    • char (single unicode characters), use 2 bytes of memory, enclose in single quotes, eg. char grade = ‘A’;
    • boolean (true and false)

  • Other types are reference types

Basic Operators

  • x = x + y is equal to x +=y
  • For division, int / int = round down int, any of non-int will give a truncated(shortened) non-int
    ( 7/2=3, 7.0 /2 = 3.5, 7/2.0=3.5, 7.0/2.0=3.5)
  • System.out.println(counter++) means print counter first, then counter++
    System.out.println(++counter) means counter++ first, then print counter

Type Casting

  • Widening primitive conversion, smaller data to larger type, can be done directly

  • Narrowing primitive conversion, larger to smaller, need use a pair of parenthesis(), may lost data, decimal portion is truncated.

Arrays and Strings

String Methods

  • .length() return total number of characters
  • .toUpperCase()/.toLowerCase() return string with uppercase/lowercase characters
  • .substring(i) return a string start from ith character (count from 0)
  • .substring(m,n) return a string start from mth and end at nth character (nth character is not included)
  • .charAt(i) return ith character of the string
  • .equals("string") return ture or false if they are equal or not

  • .split("separator") returns array of substrings, based on the separator

Array Methonds

to use the pre-weitten methods, need to add the statement

  • Arrays.equals(arr1, arr2) return true only if both value and order are same
  • Arrays.copyOfRange(source, m ,n) will return elements from index m to index (n-1), index n is not copied
  • Arrays.toString(arr) return the contents of an array
  • Arrays.sort(arr) will sort the array in ascending order
  • Arrays.binarySearch(arr, value) will return the index of the value for a sorted array, if not found, will return -i, (i-1) is the supposed index
  • .length return the size of a array, no parenthesis

Primitive Type vs Reference Type

The only 8 primitive type will store its own data
Reference type (string, array, ….) only store a reference to the data, e.g. address of the memory location of the data
